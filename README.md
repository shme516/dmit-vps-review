## 我踩过的坑，你可以跳过

买美国VPS之前，有几件事我希望有人提前告诉我：

1. **"CN2"不等于"CN2 GIA"**。前者是普通 CN2，后者是双向 GIA，延迟和稳定性差距很大，价格也差很多。买之前一定要看清楚线路描述。

2. **测速要测晚高峰**。很多商家白天的测速数据很好看，但晚上八点到十一点才是真实水平。

3. **便宜VPS的隐性成本**。迁移数据、重新配置环境、处理宕机，这些时间成本加起来往比省下的钱贵。

4. **流量超额的处理方式**。有些商家超额直接封机，有些是限速，DMIT 的规则在官网写得比较清楚，买之前看一下。
